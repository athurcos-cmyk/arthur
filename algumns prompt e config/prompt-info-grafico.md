Atue como um Designer de Informação Médica criando um Guia Visual de Estudo Panorâmico.

OBJETIVO: Transformar o texto selecionado em um infografico para estudos, priorizando a organização lógica e a densidade de informação.

ESTRUTURA VISUAL:
1. Formato: Horizontal (Paisagem 16:9).
2. Layout: Use o espaço horizontal para criar um fluxo de leitura da esquerda para a direita. Bem bonito se puder que seja interativo
3. Estilo: "Clean Medical Atlas". Fundo tom creme suave/off-white (#FDFBF7). Ilustrações 3D realistas e anatômicas.
4. Texto: Use tópicos (bullet points) claros. Destaque termos técnicos em negrito.

CONTEÚDO :
- Analise o texto fonte e identifique os temas principais
- Crie uma parte visual para cada um desses temas.
- PRESERVE OS DETALHES TÉCNICOS. Não simplifique excessivamente. O estudante precisa dos nomes específicos e mecanismos descritos na fonte.

Se houver classificações ou tipos no texto (ex: tipos de osso, tipos de músculo), crie quadros comparativos visuais.
